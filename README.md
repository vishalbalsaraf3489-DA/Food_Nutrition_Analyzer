🍎 Food Nutrition Analyzer

A Machine Learning + PySpark based Food Nutrition Analyzer that predicts the health score of food items based on their nutritional values.
This project helps analyze whether a food item is healthy or unhealthy using data-driven methods.

🚀 Project Overview

The Food Nutrition Analyzer uses PySpark ML models to analyze nutritional information such as calories, fats, carbohydrates, protein, sugar, etc., and predicts the health score of a food item.

The project demonstrates:

Data preprocessing using PySpark

Model training using PySpark ML

Feature extraction

Health score prediction

Confusion matrix evaluation
